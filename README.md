A lightweight Pomodoro CLI written in Go.

## Instal

```shell
go install github.com/GuillemotClement/pomogo@latest
```

Make sure your Go binary directory is available in your PATH.


```shell
# For Zsh users:
echo 'export PATH="$HOME/go/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## Run 

```shell
pomogo
```

## Update 

To install the latest version 

```shell
go install github.com/GuillemotClement/pomogo@latest
```
